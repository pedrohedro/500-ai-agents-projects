API in 30 seconds

  
Authenticate with a Bearer token. Here is the full money loop.

  
# 1. Sign up (get an API key + free credits)
curl -X POST {{ request.base_url }}auth/signup \
  -H 'Content-Type: application/json' \
  -d '{"email":"you@company.com"}'

# 2. Buy credits (returns a Stripe Checkout URL)
curl -X POST {{ request.base_url }}billing/checkout \
  -H 'Authorization: Bearer sk_live_...' \
  -H 'Content-Type: application/json' \
  -d '{"pack_key":"pro"}'

# 3. Call an agent (credits are metered per call)
curl -X POST {{ request.base_url }}v1/marketing/generate \
  -H 'Authorization: Bearer sk_live_...' \
  -H 'Content-Type: application/json' \
  -d '{"topic":"AI for small business","audience":"founders"}'

# 4. Check balance + usage ledger
curl {{ request.base_url }}billing/balance -H 'Authorization: Bearer sk_live_...'
